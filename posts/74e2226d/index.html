幂函数公开课教学后记

教学材料

本节PPT和Geogebra不便上传，但学案 可以上传，供大家交流使用。

设计思路

本节之前，学生已经学习了函数的概念和函数的基本性质（单调性和奇偶性）。本节就是在使用第三章所学知识，从定义域、值域、单调性、奇偶性、特殊值的角度，借助图象去总结规律，借助f(x)=x,f(x)=x2,f(x)=x−1,f(x)=x3,f(x)=x12f(x)=x,f(x)=x^2,f(x)=x^{-1},f(x)=x^3,f(x)=x^{\frac{1}{2}}f(x)=x,f(x)=x2,f(x)=x−1,f(x)=x3,f(x)=x21​归纳幂函数的概念、指数大于0、小于0、是奇数、是偶数时函数的性质。

教学设计

重点

幂函数的概念

难点

幂函数的图象和性质

教学过程

课程引入

问题1：

  1. 如果老奶奶以1𝑚/𝑠的速度过马路，那么𝑥秒后老奶奶的距离𝑦=𝑦 = y=
  2. 如果正方形的边长为𝑎，那么正方形的面积𝑆=𝑆 =S=
  3. 如果立方体的棱长为𝑏，那么立方体的体积𝑉=𝑉 =V=
  4. 如果一个正方形场地的面积为𝑆，那么这个正方形的边长𝑐=𝑐 =c=
  5. 如果精神小伙𝑡 𝑠内骑车行进了1 𝑘𝑚，那么他骑车的平均速度𝑣=𝑣 =v=

师生互动：学生得到了f(x)=x,f(x)=x2,f(x)=x−1,f(x)=x3,f(x)=x12f(x)=x,f(x)=x^2,f(x)=x^{-1},f(x)=x^3,f(x)=x^{\frac{1}{2}}f(x)=x,f(x)=x2,f(x)=x−1,f(x)=x3,f(x)=x21​五个函数解析式。

新知探究

教师：大家得到了这五个函数解析式，从形式或者长相上看，你觉得他们有什么共同特征？

师生活动：学生间相互讨论，得到的结论有：都有xxx和yyy，都是xxx的多少次方的形式。总结得到幂函数的形式——y=xα（α为常数)y=x^\alpha （\alpha为常数)y=xα（α为常数)。

问题2： 知道幂函数的概念是“长得像y=xα（α为常数)y=x^\alpha （\alpha为常数)y=xα（α为常数)的函数”后长得多像才算像呢？请你判断下面几个函数是不是幂函数。

y=x−2,y=2x2,y=(x+1)2,y=3xy=x^{-2},y=2x^2,y=(x+1)^2,y=3xy=x−2,y=2x2,y=(x+1)2,y=3x

师生活动：师生共同讨论，总结幂函数在函数形式上的特征。

  1. 底数只有自变量x
  2. 指数为常数
  3. 系数为1

再完成加强训练，巩固对概念的理解。

问题3. 请你在学案上完成填表并快速的绘制出初中所学过的f(x)=x,f(x)=x2,f(x)=x−1f(x)=x,f(x)=x^2,f(x)=x^{-1}f(x)=x,f(x)=x2,f(x)=x−1的图象。

师生活动：共同回顾作图过程及它们的定义域、值域、单调性、奇偶性、公共点。

问题4. 请你完成填表，我们共同绘制一下f(x)=x3,f(x)=x12f(x)=x^3,f(x)=x^{\frac{1}{2}}f(x)=x3,f(x)=x21​的函数图象。

师生互动：教师带领大家一起绘图（主要是考虑到时间因素，如时间允许，可以学生先自己描点连线尝试作图。

师生互动：共同分析它们的定义域、值域、单调性、奇偶性。并填写全学案上的表格。

问题5：请大家观察你得到的表格，来回答我下面的问题

  1. 这五个函数中，在第一象限内单调递增的有谁？
  2. 第一象限内单调递减的有谁？
  3. 奇函数有谁？
  4. 偶函数有谁？

师生活动：共同总结规律

  1. 指数大于0，在第一象限内单调递增；
  2. 指数小于0，在第一象限内单调递减。
  3. 指数为奇数，函数为奇函数；
  4. 指数为偶数，函数为偶函数。

概念应用

有了幂函数的概念，我们就可以借助幂函数来解决相关的问题。
其中最为直接的应用就是——比较大小。
先借助5,6,52,62,53,63,512,612,15,165,6,5^2,6^2,5^3,6^3,5^{\frac{1}{2}},6^{\frac{1}{2}},\frac{1}{5},\frac{1}{6}5,6,52,62,53,63,521​,621​,51​,61​来简单的应用。
解决例2.借助“谁变谁不变”的思想抽象得出幂函数的直接应用。
后面的两个问题分别为借助图象解决和借助单调性解决。其第一步均为确定幂函数的解析式——待定系数法。与前期所讲的“求解函数解析式”章节所呼应。

存在的问题和教学反思

  1. 画图题和上课的展示中，xxx轴和yyy轴比例尺不是1:11:11:1。会影响学生对函数图像的第一印象。
  2. 因进度问题和思路的流畅性，本节课删去了证明y=xy=\sqrt{x} y=x​单调性的例题。但结合备课组建议，在时间充分的情况下，建议保留此部分内容。
  3. 在构造幂函数比较大小的部分，讲解过于贴近答案，不够循序渐进。应引导学生在比较大小的过程中，关注“不变的是指数，变的是底数”，以此为由来构造幂函数y=xαy=x^\alphay=xα
  4. 前面讲解y=x,y=x2,y=1xy=x,y=x^2,y=\frac{1}{x}y=x,y=x2,y=x1​花费时间过长，并且这三个函数更应强调他们的整体性质。y=x3,y=x12y=x^3,y=x^{\frac{1}{2}}y=x3,y=x21​相比较而言应更详细。
  5. 增加y=x−2y=x^{-2}y=x−2的图象和性质，使性质不只局限于这5个函数。
  6. 比较大小的题目再多一些，梯度小一点。低起点，铺台阶，达目标。
